# Game of Life

## Build/run

* Requires Python3.6+ with pygame

```pip install pygame```

## Controls

* Mousewheel: Zoom
* Left Mouse: Add Cell
* Right Mouse: Remove Cell
* s: Calculate next step [Step]
* r: Randomize Board
* c: Clear Board
* p: pause/unpause simulation
* q: quit
* n: Show number of neighbours
* +/-: Speed up/slow down animation
